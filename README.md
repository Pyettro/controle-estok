# Controle de Estoque

Aplicação React + Vite para controle de estoque com shadcn-ui e React Query.

## Deploy (GitHub Pages)
- URL pública: https://pyettro.github.io/controle-estok/
- Branch: `main`
- Pasta publicada: `docs/` (gerada por `npm run build`)
- Workflow: `.github/workflows/pages.yml`

## Como rodar localmente
```sh
npm ci
npm run dev
```

Build para Pages:
```sh
npm run build   # gera docs/
```

## Tech stack
- React 18 + Vite + TypeScript
- Tailwind CSS + shadcn-ui
- React Router (HashRouter) + React Query
